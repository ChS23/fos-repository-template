# Команда проекта

Сведения об авторах, ролях и вкладе участников разработки фонда оценочных средств.

| Участник | Организация | Роль | Зона ответственности | Контакты / профиль |
|---|---|---|---|---|
|  |  |  |  |  |

Раздел заполняется коллективом разработчиков дисциплины. Для каждого участника указываются организация, роль (методолог, постановщик задач, эксперт по КРМ, разработчик КИМ, эксперт по содержанию) и зона ответственности.

## Вклад участников

Вклад описывается по принципу CRediT или в свободной форме: концепция, методология, разработка контрольно-измерительных материалов, программная реализация проверок, экспертиза содержания, редактирование, управление проектом.

## Ответственные за актуализацию

За ежегодную сверку нормативной базы ([стандарты](../resources/other/README.md)) и инструментального контура ([программные средства](../resources/software/python-libs/README.md)) назначается ответственный из числа участников команды. Дата сверки фиксируется в соответствующих карточках ресурсов.
