# Система итогового оценивания

Оценивание построено по балльно-рейтинговой системе. Итоговый балл формируется из результатов текущего контроля по модулям и проектной работы; промежуточная аттестация проводится в форме зачёта с защитой проекта.

## 1. Структура итогового балла

Каждый контрольно-измерительный материал оценивается по 100-балльной шкале (критерии приведены в файле соответствующего КИМ). Итоговый балл дисциплины рассчитывается с весами:

| Оценочное средство | Индикаторы | Вес |
|---|---|:--:|
| КИМ-1.5. Feasibility Study | LC-1.1 | 15 % |
| КИМ-2.5. Техническое задание и требования | LC-1.3, LC-1.2 | 15 % |
| КИМ-3.4. Архитектура | LC-3.1, ОПК-4.1 | 15 % |
| КИМ-4.1. Верификация и валидация | LC-3.1, LC-4.1 | 15 % |
| Проектная работа | ОПК-4.1, LC-1.1, LC-1.2, LC-1.3, LC-3.1, LC-4.1 | 40 % |

Итоговый балл = 0,15 · (КИМ-1.5 + КИМ-2.5 + КИМ-3.4 + КИМ-4.1) + 0,40 · Проект.

## 2. Условия зачёта

Зачёт выставляется при одновременном выполнении условий:

- итоговый балл не ниже 55 из 100;
- каждый КИМ модуля сдан с результатом не ниже 55 баллов по своей шкале;
- проектная работа представлена и защищена на индивидуальном уровне.

Невыполнение любого условия означает незачёт независимо от суммы баллов. Правило «артефакт без защиты не оценивается» действует для каждого оценочного средства.

## 3. Перевод в шкалу оценок

| Итоговый балл | Оценка |
|:--:|---|
| 85–100 | отлично (зачтено) |
| 70–84 | хорошо (зачтено) |
| 55–69 | удовлетворительно (зачтено) |
| менее 55 | не зачтено |

## 4. Сроки и пересдача

Сроки сдачи КИМ устанавливаются календарным планом дисциплины. Порядок пересдачи определяется локальными нормативными актами образовательной организации.

## 5. Согласование

Веса и пороги, приведённые в разделах 1–3, подлежат согласованию с учебным планом и локальными нормативными актами образовательной организации до начала семестра.
